# Amazon Sales Analysis (SQL Project)

## 📌 Problem Statement
Analyze sales data using SQL to identify trends and insights.

## 🛠 Tools Used
- SQL

## 📂 Project Structure
- sql/ → SQL queries

## 📊 Key Insights
- Are high-value customers driving most of the revenue?
- Which products are underperforming in sales?
- Are discounts improving sales or reducing revenue?
- Are sales increasing over time (monthly trend)?
- Which sellers contribute most to revenue?
- Who are the top 5 customers by revenue?
- Which products have high discounts but low revenue?
- How does order status (completed vs pending) impact sales?

## 📂 Dataset

- Source: Kaggle  
- Dataset: Amazon Sales Dataset  
- Link: https://www.kaggle.com/datasets/rohiteng/amazon-sales-dataset  

This dataset contains 100,000 synthetic Amazon-style e-commerce transactions, including customer details, product information, pricing, discounts, payment methods, and order status.

> Note: Dataset is not included in this repository due to size constraints. Please download it from the above link.

## 📊 Dataset Features

- Order Details: OrderID, OrderDate, OrderStatus  
- Customer Info: CustomerID, CustomerName, Location  
- Product Info: ProductID, ProductName, Category, Brand  
- Financial Metrics: UnitPrice, Discount, Tax, ShippingCost, TotalAmount  
- Payment: PaymentMethod

SQL analysis helps in making data-driven business decisions.
